維新回天・竜馬伝！／ザ・クラシック

宙組／宝塚大劇場2006年11/3～12/12・東京宝塚劇場1/2～2/12

■ カレン
　 Date: 2006-11-13 (Mon)

ＪＩＭＭＹさん皆さんお久しぶりです。

１１月１０日　宙組公演「維新回天・竜馬伝！」を観劇してきました。
先ず今回のようなぜいたくな組み合わせはあり得ないとおもいます。
雪組から　貴城　けい　の最初で最後のトップを迎え、花組から、蘭寿　とむ　月組から　北翔　海莉とこれからどんどんよくなる人達を得、またどんどん伸びてきている宙組ならではの大型男役３人（悠末　ひろ七帆　ひかる　十輝　いりす）が揃って大変見応えのある公演です。

　貴城　けい　の最後作品です。貴城は大変気合いが入っていました。　
竜馬をくだけた感じにつくっていましたが、元々日本物のよく似合ううまい人だけに、きつちりときれいに演じていました。着物の着こなしや少しくずれた鬘のつきもよかったです。　竜馬が、新しい日本に変えていこうと考えていたことがよく表現できていました。

蘭寿　は最後の将軍を知的に演じて好感がもてました。歌もよかったです。あの鬘は似合うのが、難しいとおもいますが、「くらわんか」で青天をしたから、違和感がなかったのでしょう。

北翔海莉　は芝居がうまくなりましたね。　女に化けて、鳥追いのかっこうしている場面は面白かったです。男が女に化けている感じがうまくだせていました。
北翔　は声がはっきりしているのが、なによりです。歌は期待どうり良かったです。

武市　半平太　の　悠末はきれいでしたが、出る場面が少ないからか役がらがあまりわからなかったです。　十輝　の高杉晋作　も同様でした。
七帆　ひかるの陸奥　宗光　は竜馬　と一緒にでる場面が多くすっきりと海援隊の隊士を演じて、やる気をかんじさせました。

お竜の　紫城　るい　は身のこなしが粋で、声も良くこれまでで、一番よい出来ではないでしょうか。
少し気が強いが、竜馬に対する愛情が深いお竜をよく演じられていました。

この芝居は石田　昌也　さん　の作品だけに　西郷隆盛に品のないセリフをしゃべらせたりしますが、全体にすっきりとした分かり易い面白い適材適所のよい作品だと思いました。

ショー　「ザ・クラッシック」は幕開きの舞台が、きらきらとまるで神戸　の　ルミナリエを意識しているようでした。
宝塚の醍醐味を感じます。
最初オケボックスより　蘭寿　が飛び出し歌い踊ります。そこへ　ヘヤースタイルに工夫を凝らした、北翔　と大型３人組が加わり、「アイラブ　ショパン」と歌います。この主題歌は最近になく覚えやすく耳に残り楽しい歌です。

貴城　のノーブルな持ち味から、クラッシックな歌やダンスが続くので、やや平板な感じもしますが、
ショパンの貴城とジョルジュ・サンドの　大和　悠河に　運命役の　蘭寿　が絡む場面は色が違います。
私は　老貴婦人　の　紫城　るい　に　老執事　の　悠末、十輝、七帆　の３人がからんで歌う場面が面白いし、明るいので好きでした。

終わりの　トップ二人のディエットダンスはすてきですし、貴城　が一人で別れの歌を歌うのもよかったです。

これで今年三人目のトップスターの宝塚大劇場の最後の芝居となりましたね。




■ JIMMY
　 Date: 2006-11-03 (Fri)
